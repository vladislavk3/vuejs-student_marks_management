# vue-admin-panel

**Complex Data Visualization**

## Features

- Vue.js
- ElementUI
- Vue-Router
- Vuex
- Crypto-js
- echarts
- FontAwsome Icons
- Markdown Support
- Browser-Sync

## Development

```bash
npm install # Install everything you need
npm start | npm run dev # Start up Browser-Sync to automatically refresh when changes are made
npm build # Build production
```
